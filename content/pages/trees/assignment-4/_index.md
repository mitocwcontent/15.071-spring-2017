This data comes from the <span style="text-decoration: underline;"><a href="http://archive.ics.uci.edu/ml/datasets/Adult">UCI Machine Learning Repository</a></span>.</p> <p>The file <span style="text-decoration: underline;"><a href="./resolveuid/23bdcd0033c04ce7b2ea204f161b30e0">census (CSV -
